This project depends also on:
- rosjava_tf_core

This project can be found on http://code.google.com/p/rosjava-tf/

Building rosjava_tf_core:
1) import in eclipse
2) install vecmath library: sudo apt-get install libvecmath-java
3) download jgrapht library from http://jgrapht.org/
4) fix dependencies
5) modified the file TransformFactory.java, TransformBroadcaster.java, TransformListener.java
